# 사주번호 운세 계산기

생년월일을 기반으로 사주번호와 해운번호를 계산하여 나와 잘 맞는 장소, 번호, 궁합 정보를 제공하는 웹 애플리케이션입니다.

## 주요 기능

### 📊 운세 계산
- **사주번호**: 생년월일 기반 평생 운명수 (비중 80%)
- **이름 번호**: 각 글자의 운명수 (비중 20%)
- **해운번호**: 매년 변하는 올해 운세 번호

### 🎯 매칭 정보
- 행운의 숫자와 지양할 숫자
- 지역 초성 매칭 (중복 강조)
- 추천 지형 및 지역
- 궁합이 좋은 사주번호 (중복 강조)
- 특수 사주 조합 분석

### ✨ 특징
- 양력/음력 모두 지원
- 4개 사주번호 통합 분석
- 중복 초성/궁합 강조 표시
- 상세한 풀이와 설명
- 반응형 디자인 (모바일 최적화)
- 다크모드 자동 지원

## 기술 스택

- HTML5
- CSS3 (Flexbox, Grid, Gradient)
- Vanilla JavaScript (ES6+)
- 타로 카드 기반 사주 해석
